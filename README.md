# Vehicle-Rental-System

**Project Description**
The Vehicle Rental System is a full-stack web application developed using the MERN Stack (MongoDB, Express.js, React.js, Node.js). It allows users to browse available vehicles, make bookings, and manage rentals efficiently through a user-friendly interface.

**Features**
🔐 User Authentication
Secure login and registration system
🚘 Vehicle Management
View available vehicles with details
Filter and search vehicles
📅 Booking System
Book vehicles for specific dates
Real-time availability tracking
💳 Payment Processing
Simulated payment and booking confirmation
🧾 Booking History
Users can view past and current bookings
🛠️ Admin Panel
Add, update, and delete vehicles
Manage bookings and users

**Technologies Used**
Frontend: React.js, HTML, CSS, JavaScript
Backend: Node.js, Express.js
Database: MongoDB
Other Tools: REST APIs, JSON, JWT (for authentication)

**System Modules**
**User Module**
    Registration & Login
    Vehicle browsing and booking
**Admin Module**
    Manage vehicles
    Monitor bookings and users
**Booking Module**
    Handles reservation and availability
    Stores booking details in database
